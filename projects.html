Here are the projects I worked on!

Gulf of Mexico, is the perfect programming language I ever worked on, it is special in the way that it only features 4 different types of declarations, the array indexing starts at 1 and not from 0, and amongst the boolean values there is "true", "false", and "maybe".

Tools used: Brain, Internet, not Java

A High school project that helped our school manage underage students who wished to exit school grounds during breaks digitally, with only minimal supervision. It was done using a chip reader at the entrance, and dedicated nfc exit cards were handed for the students who are permitted.

Tools used: Java

Godot is a popular beginner friendly game engine that support 2d and 3d game development, it has many built in features that help animation, rigging, and level design. It is highly intuitive, and has a lot of tutorials for people just starting out game development.

Tools used: C++, Java

Barotrauma is a survival horror multiplayer game, in it you manage your crew of incompetent crewmates while trying to keep the ship moving, and to keep intruders out. This project was to make modding easier,and for people who want extra freedom when writing scripts in the game using Lua.

Tools used: C#, C, C++

World of Warcraft is a massively multiplayer game (MMO), that amongst its many features, has fishing. This project was developed to make fishing automatic, intuitive, and require zero attention. It listens to in-game audio, looks for sprites and splices it with previous screenshots to find the fishing bobber, while also being able to apply hooks, and baits to the fishing pole.

Tools used: C#
